# 🎬 Netflix Movies & TV Shows Data Analysis Dashboard

An interactive **Netflix Data Analysis & Movie Recommendation System** built using **Python, Streamlit, and Machine Learning**.

This project analyzes Netflix content and provides visual insights along with a content-based recommendation system.

## 🚀 Features

✅ Interactive Dashboard using Streamlit
✅ Netflix Dataset Analysis
✅ Movie & TV Show Filtering
✅ Global Movie Search (OMDB API)
✅ Content-Based Recommendation System
✅ Dynamic Visualizations
✅ Posters fetched using API
✅ Real-time Data Filtering

## 📊 Dashboard Includes

* Movies vs TV Shows Distribution
* Rating Analysis
* Top Directors
* Genre Analysis
* Country-wise Content
* Release Year Trends
* Recommendation Engine

## 🧠 Recommendation System

The recommendation system uses:
* TF-IDF Vectorization
* Cosine Similarity
* Content-based filtering using:

  * Genre
  * Description

It suggests similar movies based on user selection.
